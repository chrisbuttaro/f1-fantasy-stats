## UI Components

| What you see | Name |
|---|---|
| Ranked list of drivers or constructors | Ranking table |
| Row that expands on click | Expanded row / chart panel |
| Line graph showing points per race | Race chart |
| Fixed bar at the bottom with picks and budget | Team bar |
| Driver/constructor cards in the team bar | Chips |
